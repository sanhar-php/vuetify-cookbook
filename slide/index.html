Why Vuetify?

Material Design

  • Material Design 是 Google 发布的全平台设计规范
  • 完美支持规范中的所有色彩 和 图标

语义化工具箱

  • 像说话一样的编写组件，让 coding 变得更加顺畅

强大的支持

  • 响应式布局 + 活跃的社区 + 持续的更新

Cookbook(01-setup)

git branch 01-setup && git checkout 01-setup && git add -A && git commit -m "Cookbook(01-setup)"

参考 快速入门 创建全新项目 或 引入到现存项目

创建 vuetify-cookbook 项目

$ vue init vuetifyjs/webpack vuetify-cookbook //创建项目选用的参数

启动项目

npm run dev

npm i webpack-dev-server --save-dev //添加遗漏的插件

浏览器打开

http://localhost:8080
